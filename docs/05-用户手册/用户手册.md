# ISP-Agent 用户手册

## 1. 产品介绍

### 1.1 什么是ISP-Agent

ISP-Agent 是一款基于LLM的ISP图像/视频处理助手，帮助ISP工程师进行图像质量分析、参数诊断和调优建议。

### 1.2 主要功能

- 📊 图像质量分析
- 🔍 ISP参数诊断
- 🎬 视频处理分析
- 💡 智能调优建议
- 🗣️ 自然语言对话

---

## 2. 环境准备

### 2.1 系统要求

| 项目 | 要求 |
|------|------|
| 操作系统 | macOS / Linux / Windows (WSL) |
| Python | >= 3.9 |
| 内存 | >= 8GB |
| 硬盘 | >= 10GB |

### 2.2 安装步骤

```bash
# 1. 克隆或下载项目
cd ~/isp-agent

# 2. 创建虚拟环境
python -m venv venv

# 3. 激活虚拟环境
source venv/bin/activate  # macOS/Linux
# 或
venv\Scripts\activate     # Windows

# 4. 安装依赖
pip install -r requirements.txt

# 5. 配置环境变量
export OPENAI_API_KEY="sk-your-key-here"

# 6. 运行
python src/main.py --help
```

## 3. 配置说明

### 3.1 配置文件

编辑 `config/default.yaml`:

```yaml
llm:
  provider: "openai"  # 可选: openai, anthropic, minimax
  model: "gpt-4o"
  api_key: "${OPENAI_API_KEY}"  # 或直接填写密钥
  temperature: 0.7

tools:
  image:
    max_size_mb: 50
  video:
    max_size_mb: 100
    max_duration_seconds: 300
```

### 3.2 API密钥配置

**OpenAI:**
```bash
export OPENAI_API_KEY="sk-xxx"
```

**Anthropic:**
```bash
export ANTHROPIC_API_KEY="sk-ant-xxx"
```

**MiniMax:**
```bash
export MINIMAX_API_KEY="your-key"
```

## 4. 使用方法

### 4.1 命令行模式

```bash
# 对话模式
python src/main.py --chat

# 分析图像
python src/main.py --image test.jpg

# 分析视频
python src/main.py --video test.mp4
```

### 4.2 对话示例

```
你: 分析一下这张图片
ISP-Agent: (自动分析图像并返回结果)

你: 这张照片噪点很多，怎么处理？
ISP-Agent: 建议使用降噪功能，我来帮你处理...

你: 曝光有点问题，怎么调整ISP参数？
ISP-Agent: 根据分析结果，建议调整以下参数...
```

## 6. 功能说明

### 6.1 图像分析

- **基本信息**: 分辨率、格式、大小
- **EXIF数据**: 相机、ISO、光圈、快门
- **直方图**: RGB和亮度分布
- **质量分析**: 清晰度、噪声、曝光、色彩

### 6.2 图像处理

- **降噪**: 高ISO降噪处理
- **锐化**: USM锐化增强
- **调整**: 色彩、对比度

### 6.3 视频分析

- **基本信息**: 编码、分辨率、帧率
- **抽帧**: 提取关键帧

### 6.4 ISP问答

- 曝光参数调整
- 白平衡优化
- 降噪参数设置
- 锐化强度控制

## 7. 常见问题

### Q1: 图像分析很慢怎么办？
A: 建议使用1080P以下图像，4K图像处理时间较长。

### Q2: API调用失败怎么办？
A: 检查网络连接和API密钥配置是否正确。

### Q3: 不支持的文件格式？
A: 转换为支持的格式后再处理。
