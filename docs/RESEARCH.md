# ISP-Agent 研究报告 v0.4

## 调研日期
2026-03-27

## 业界最新技术趋势

### 1. AI-ISP (全AI驱动的ISP Pipeline)
- **Visionary.ai**: 世界首个全AI驱动的ISP，使用神经网络替代传统ISP模块
  - AI降噪：可在0.1 lux极低光照下工作
  - AI HDR融合：实时传感器数据增强
  - 软件定义成像管线，可在NPU上运行
  
- **Ambarella CV7**: 4nm AI ISP芯片
  - 神经网络增强的ISP，低光/HDR性能提升100倍
  - 支持多路8K视频流处理
  - 针对汽车ADAS优化

### 2. 无参考图像质量评估 (NR-IQA) 前沿
- **BRISQUE** (Blind/Referenceless Image Spatial Quality Evaluator)
  - 使用自然场景统计(NSS)特征
  - 不需要参考图像
  - 支持向量回归(SVR)进行质量预测
  
- **NIQE** (Naturalness Image Quality Evaluator)
  - 基于自然图像统计模型
  - 完全无参考
  - 对失真类型不敏感

- **Deep Learning based**: 
  - CNN/RNN-based models
  - TOPIQ, Musiq, TriQE等最新模型
  - 但需要大量训练数据

### 3. 车载图像质量标准
- **ISO 16505** (Automotive摄像头标准)
- **AEC-Q100** (汽车电子可靠性)
- **IEEE 802.3** (车载以太网)
- ADAS前视摄像头推荐规格：
  - 分辨率: 1920x1080 (最低) / 3840x2160 (推荐)
  - 帧率: 30fps (最低) / 60fps (推荐)
  - 动态范围: 100dB+ (HDR)
  - 视场角: 100°-180°

### 4. ISP调优知识库建设
- 基于规则的专家系统
- 场景识别 → 参数推荐
- 问题诊断 → 解决方案映射
- 自动调参算法（强化学习方向）

## 我们的定位与机会

### 差异化优势
1. **开源+API优先** - 可集成到任何工作流
2. **车载专用** - 针对ADAS/自动驾驶场景优化
3. **本地运行** - 无需云端，保护数据隐私
4. **知识库+AI** - 不仅仅是工具，更是专家系统

### 技术差距
- AI质量评分使用启发式方法，非ML模型
- HDR算法较为基础
- RAW处理依赖rawpy，功能有限
- 知识库规则数量有限

## v0.4 改进方向

1. **BRISQUE-style NSS特征提取** - 实现基于自然场景统计的质量评估
2. **多 Tone Mapping 算法** - 支持Reinhard/Mantiuk/Drago等
3. **增强车载分析** - 对标ISO 16505标准
4. **知识库扩展** - 增加更多ISP模块诊断规则
5. **视频质量分析** - 运动模糊检测、帧率稳定性分析
6. **测试覆盖率提升** - 关键模块单元测试

## v1.0 目标（对标世界一流）

1. **ML-based IQA** - 可选的深度学习质量评估模型
2. **完整ISP Pipeline模拟** - 可配置的参数调整
3. **自动驾驶感知质量评估** -目标检测、车道线检测等任务质量评估
4. **多摄像头同步分析** - 环视拼接质量评估
5. **调参自动化** - 基于分析结果的参数推荐
6. **Web UI** - 友好的图形界面
