## 使用方式

### CLI
```bash
python -m src.cli_enhanced analyze image.jpg
python -m src.cli_enhanced automotive image.jpg --scene adas_front
python -m src.cli_enhanced quality image.jpg
python -m src.cli_enhanced pipeline --explain Demosaic
```

### Web API
```bash
# 启动服务
python -m src.api

# 调用API
curl -X POST -F "file=@image.jpg" http://localhost:5000/api/analyze
```

### Docker
```bash
docker-compose up -d
```
