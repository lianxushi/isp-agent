## 五、使用说明

### 5.1 环境配置

```bash
# 克隆项目
cd ~/.openclaw/workspace/isp-agent

# 激活虚拟环境
source venv/bin/activate

# 配置API密钥
export MINIMAX_API_KEY="your-api-key"
```

### 5.2 使用命令

```bash
# 对话模式
python src/main.py chat

# 分析图像
python src/main.py analyze your_image.jpg

# 分析视频
python src/main.py analyze your_video.mp4

# 图像处理
python src/main.py process your_image.jpg denoise
```
