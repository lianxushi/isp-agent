# ISP-Agent 架构设计文档

## 1. 系统概述

### 1.1 设计目标
- 模块化架构，便于扩展
- 解耦核心逻辑与LLM调用
- 支持多种图像/视频处理后端

### 1.2 技术选型

| 层级 | 技术栈 | 说明 |
|------|--------|------|
| 大脑 (LLM) | MiniMax API | 中文理解能力强，性价比高 |
| 图像处理 | OpenCV | 通用图像处理 |
| RAW处理 | rawpy | RAW格式解析 |
| 视频处理 | FFmpeg + OpenCV | 视频编解码与分析 |
| 框架 | FastAPI | 轻量级Web服务 |
| 部署 | 本地运行 | Python环境直接启动 |

## 3. 核心模块设计

### 3.1 ImageAnalyzer (图像分析器)

```python
class ImageAnalyzer:
    def analyze_basic_info(self, image_path) -> dict
    def analyze_histogram(self, image_path) -> dict
    def analyze_dynamic_range(self, image_path) -> dict
    def analyze_noise(self, image_path) -> dict
    def analyze_color(self, image_path) -> dict
    def apply_denoise(self, image_path, method) -> str
    def apply_sharpen(self, image_path, amount) -> str
    def apply_color_adjust(self, image_path, temp, tint) -> str
```

### 3.2 VideoAnalyzer (视频分析器)

```python
class VideoAnalyzer:
    def get_video_info(self, video_path) -> dict
    def extract_frames(self, video_path, interval) -> list
    def analyze_stability(self, video_path) -> dict
    def analyze_quality(self, video_path) -> dict
```

### 3.3 QAEngine (智能问答引擎)

```python
class QAEngine:
    def __init__(self, llm_client)
    def chat(self, user_input, context) -> str
    def generate_suggestion(self, analysis_result) -> str
    def explain_isp_param(self, param_name) -> str
```

### 3.4 LLmClient (LLM接入)

```python
class LLMClient:
    def __init__(self, api_key, base_url, model)
    def chat(self, messages, temperature) -> str
    def chat_with_image(self, image_base64, prompt) -> str
```

## 4. 数据流设计

### 4.1 图像分析流程

```
用户上传图片
    ↓
文件校验 (格式、大小)
    ↓
ImageAnalyzer 分析
    ↓
结构化结果 (JSON)
    ↓
[可选] LLM 解读 + 建议
    ↓
输出报告
```

### 4.2 智能问答流程

```
用户自然语言输入
    ↓
意图识别 + 参数提取
    ↓
[需要分析?] → 调用 ImageAnalyzer
    ↓
构建 Prompt
    ↓
LLM 响应
    ↓
格式化输出
```

## 5. 配置管理

```yaml
# config/settings.yaml
llm:
  provider: minimax
  api_key: ${MINIMAX_API_KEY}
  base_url: https://api.minimax.chat/v1
  model: abab6.5s-chat
  temperature: 0.7

image:
  max_size: 50MB
  supported_formats: [jpg, jpeg, png, tiff, dng, raw]
  default_quality: 95

video:
  max_size: 500MB
  supported_formats: [mp4, mov, avi, mkv]
  max_duration: 300

logging:
  level: INFO
  file: logs/isp-agent.log
```

## 7. API接口设计

### 7.1 REST API

| 方法 | 路径 | 描述 |
|------|------|------|
| POST | /analyze/image | 分析图像 |
| POST | /analyze/video | 分析视频 |
| POST | /chat | 智能问答 |
| POST | /process/denoise | 图像降噪 |
| POST | /process/sharpen | 图像锐化 |
| GET | /health | 健康检查 |

### 7.2 CLI命令

```bash
# 图像分析
isp-agent analyze image.jpg

# 视频分析  
isp-agent analyze video.mp4

# 智能问答
isp-agent chat "这张图有什么问题？"

# 图像处理
isp-agent process denoise image.jpg
```
