## 五、数据格式规范

### 5.1 Comp12 RAW 格式

```python
# Comp12 → Raw16 转换
def comp12_to_raw16(comp12_data):
    """12-bit Comp12 转 16-bit Raw"""
    raw16 = np.zeros_like(comp12_data, dtype=np.uint16)
    raw16 = comp12_data.astype(np.uint16)  # 低12位
    # 高4位补0 (已默认)
    return raw16

# RGB Pattern 识别
PATTERNS = {
    'RGGB': (0, 1, 1, 2),
    'BGGR': (1, 0, 2, 1),
    'GRBG': (0, 2, 1, 1),
    'GBRG': (1, 2, 0, 1),
}
```

### 5.2 BRISQUE 指标

```
BRISQUE Score: 0-100 (越低越好)
  - 0-20: 优秀 (Excellent)
  - 20-40: 良好 (Good)
  - 40-60: 中等 (Fair)
  - 60-80: 较差 (Poor)
  - 80-100: 很差 (Bad)
```
