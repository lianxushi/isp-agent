## 一、产品概述

### 1.1 产品定位

**ISP-Agent (OpenClaw Edition)** 是一款基于 **OpenClaw 平台**的 ISP 领域专业智能助手，通过自然语言交互帮助工程师完成图像质量分析、ISP 参数诊断、调参优化建议等任务。

**核心价值**:
- 降低 ISP 调参门槛，新人可通过对话获得专业建议
- 自动化分析 + AI 建议，提升调试效率
- 沉淀专家经验，形成可传承的知识库

## 三、用户与场景

### 3.1 目标用户

| 用户群体 | 画像 | 痛点 | 需求 |
|----------|------|------|------|
| **A: ISP 算法工程师** | 3-5年经验 | 调参依赖经验，周期长 | 快速定位问题 |
| **B: 相机调优工程师** | 1-3年经验 | 场景配置复杂 | 场景化模板 |
| **C: 自动驾驶视觉工程师** | 5年+经验 | 车载要求严格 | 标准合规检查 |

### 3.2 核心场景

#### 场景 1: 图像质量问题诊断
```
触发: 工程师发现图像有问题
流程: 上传图像 → AI 分析 → 问题定位 → 调参建议
输出: 问题诊断报告 + 调参建议
```

#### 场景 2: 新场景调参辅助
```
触发: 遇到新场景 (如隧道入口)
流程: 描述场景 → 检索模板 → 生成参数 → 微调验证
输出: 初始参数 + 注意事项
```

#### 场景 3: 自动驾驶摄像头评估
```
触发: 评估新摄像头/供应商
流程: 上传测试素材 → 执行测试 → 生成报告
输出: 评估报告 + 评分
```

## 四、技术选型

| 组件 | 选择 | 理由 |
|------|------|------|
| **平台** | OpenClaw | 插件化成熟，多通道支持 |
| **LLM** | MiniMax-M2.5 | 性价比高，支持 Function Call |
| **图像处理** | Python + OpenCV | 成熟稳定 |
| **视频处理** | FFmpeg + OpenCV | 全格式支持 |
| **知识库** | 向量数据库 | OpenClaw 原生支持 |
| **部署** | Docker | 一键部署 |

## 五、开发里程碑

### Phase 1: MVP (2周)
- ISP Plugin 框架搭建
- 基础图像分析 (信息提取、直方图)
- 基础知识问答
- 单通道接入 (飞书)

### Phase 2: v1.0 (4周)
- 完整图像分析 (噪声、色彩、清晰度)
- 视频分析
- 知识库集成
- 多通道接入

### Phase 3: v1.x (持续)
- AI 质量评分
- 车载场景增强
- 高级分析功能
