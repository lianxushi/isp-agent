## 一、项目概述

### 1.1 项目背景

ISP（Image Signal Processor）算法工程师在日常工作中面临以下痛点：

| 痛点 | 现状 | 期望 |
|------|------|------|
| 图像质量评估 | 依赖 Imatest 等工具，人工操作繁琐 | 自动化批量分析 |
| 版本对比 | 手动截图对比，无法量化差异 | 自动生成对比报告 |
| 参数调优 | 依赖经验，迭代周期长 | AI 智能建议 |
| 知识积累 | 经验分散在个人笔记中 | 结构化知识库 |
| 文档编写 | 手动撰写测试报告 | 自动生成规范文档 |

### 1.2 项目目标

**ISP-Agent** 是一款基于 LLM 的 ISP 领域智能助手，通过自然语言交互帮助工程师完成：

- ✅ 图像质量自动化分析（BRISQUE/HDR/色彩/噪声）
- ✅ ISP 版本对比自动化（Comp12 RAW → 对比报告）
- ✅ 智能调参建议（基于分析结果的 AI 建议）
- ✅ 中文自然语言 ISP 问答
- ✅ 测试报告自动生成

## 二、核心功能

### 2.1 ISP 版本对比框架（核心功能）

#### 2.1.1 功能描述

自动化批量对比不同 ISP 版本的图像质量差异，支持多维度量化分析。

#### 2.1.2 输入数据

| 数据类型 | 格式 | 说明 |
|----------|------|------|
| Comp12 RAW | 12-bit 原始数据 | RGGB/BGGR/GRBG/GBRG 格式 |
| CModel ISP | 处理后图像 | ISP Pipeline 输出 |
| 参考图像 | PNG/JPEG | ground truth |

#### 2.1.3 核心指标

| 指标 | 说明 | 优先级 |
|------|------|--------|
| **BRISQUE** | 盲图像质量评分 (0-100) | P0 |
| **HDR动态范围** | 动态范围 (dB) | P0 |
| **色彩准确性** | ΔE 色差 | P1 |
| **噪声水平** | SNR (dB) | P1 |
| **边缘增强 (EE)** | 锐度评估 | P2 |
| **局部色调映射 (LTM)** | 局部对比度 | P2 |
| **ISO16505 合规** | 标准化评估 | P1 |

#### 2.1.4 输出

- **HTML 对比报告**：可视化图表 + 量化数据
- **调参建议**：基于 AI 分析的优化方向
- **历史追踪**：历次对比记录查询

### 2.2 图像质量分析

#### 2.2.1 直方图分析
- RGB/HSV 通道分布
- 动态范围计算
- 曝光统计

#### 2.2.2 噪声分析
- 噪声水平估计
- 噪声频谱分析
- 时域噪声 vs 空间域噪声

#### 2.2.3 RAW 域处理
- Bayer 插值
- 坏点检测与校正
- 暗电流校正

---

### 2.3 ISP 参数诊断

| 参数 | 诊断内容 |
|------|----------|
| Gamma | Gamma 曲线评估 |
| EE | 边缘增强强度分析 |
| LTM | 局部色调映射效果 |
| AWB | 自动白平衡准确性 |

### 2.4 智能问答

#### 2.4.1 功能描述
中文自然语言 ISP 领域问答，支持：
- 技术问题咨询
- 调参方案建议

#### 2.4.2 问答引擎

- **上下文记忆**: 支持多轮对话
